# Resume Parser using NLP

An NLP-based Resume Parser that extracts key information from PDF
resumes and converts unstructured data into structured JSON output,
similar to real-world ATS systems.

## Features

-   Name extraction using rule-based logic + spaCy NLP
-   Email and phone number extraction using Regex
-   Education and skill extraction with duplicate handling
-   Outputs clean, structured JSON data

## Tech Stack

Python, spaCy, Regex, pdfplumber, Google Colab

## How to Run

1.  Open the notebook in Google Colab
2.  Upload a resume PDF
3.  Run all cells
4.  Get extracted data in JSON format
